# Product Overview

A production-ready Next.js Docker example application demonstrating how to containerize Next.js apps using standalone output mode. Serves as a reference/starter for deploying Next.js in Docker with optimized multi-stage builds, supporting both Node.js and Bun runtimes.

The app itself is a landing page showcasing the Docker setup with quick-start instructions and links to Next.js/Docker resources.
